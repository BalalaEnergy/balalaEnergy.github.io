更改ssh的连接端口

2021年05月24日 172 字 大概 1 分钟

Vim /etc/ssh/sshd_config

找到port 22 在下面加上要更改的端口

在可以通过新端口访问前，不要关闭现有连接！！！！

默认情况下，SELinux 只允许端口 22 用于 SSH，现在需要做的是通过 SELinux 启用新创建的端口
semanage port -a -t ssh_port_t -p tcp 10252
如果要移除该端口，使用 -d参数
semanage port -d -t ssh_port_t -p tcp 10252

如果未安装semanage，执行yum -y install policycoreutils-python

允许新端口通过防火墙
firewall-cmd –permanent –zone=public –add-port=10252/tcp

重启防火墙
firewall-cmd –reload

重启防火墙服务
systemctl restart sshd.service

查看新端口是否生效
systemctl status sshd.service

使用新端口进行连接
