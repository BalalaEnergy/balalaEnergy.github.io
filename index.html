更改ssh的连接端口
发表于2021-05-24|随手记
Vim /etc/ssh/sshd_config 找到port 22 在下面加上要更改的端口 在可以通过新端口访问前，不要关闭现有连接！！！！ 默认情况下，SELinux 只允许端口 22 用于 SSH，现在需要做的是通过 SELinux 启用新创建的端口semanage port -a -t ssh_port_t -p tcp 10252如果要移除该端口，使用 -d参数semanage port -d -t ssh_port_t -p tcp 10252 如果未安装semanage，执行yum -y install policycoreutils-python 允许新端口通过防火墙firewall-cmd –permanent –zone=public –add-port=10252/tcp 重启防火墙firewall-cmd –reload 重启防火墙服务systemctl restart sshd.service 查看新端口是否生效systemctl status sshd.service 使用新端口进行连接    
Centos7 防火墙相关操作
发表于2021-05-24|随手记
查看zone名称：firewall-cmd –get-active-zones 开启端口（永久）firewall-cmd –zone=public –add-port=6379/tcp –permanent 重启防火墙firewall-cmd –reload 查看端口号是否开启firewall-cmd –query-port=6379/tcp 查看防火墙状态firewall-cmd –state 停止防火墙systemctl stop firewalld.service 禁止防火墙开机自启systemctl disable firewalld.service 启动防火墙systemctl start firewalld.service
