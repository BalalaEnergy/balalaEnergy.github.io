Windo配置git全局代理

git config –global https.proxy socks5://127.0.0.1:1080

如何将github上的public项目fork为private项目
2021年05月24日 59 字 大概 1 分钟

1.先在GitHub上创建一个private项目
2.将public项目clone到本地
git clone –bare https://github.com/exampleuser/public-repo.git
3.进入到刚刚clone的项目里
cd public-repo.git
4.push到刚刚创建的private项目中
git push –mirror https://github.com/yourname/private-repo.git
